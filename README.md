## COMMIT CONVENTION
- ➕ **[ADD]** : TIL 문서나 부수적 코드 추가
- ✅ **[MOD]** : TIL 문서 및 내부 파일 수정
- 🗑 **[DEL]** : 쓸모없는 코드 및 파일 삭제
- ✏️ **[CORRECT]** : 문법 오류 해결, 타입 변경, 이름 변경 등의 작은 수정
- 📄 **[DOCS]** : README 등의 부수적 문서 개정
- 🚚 **[MOVE]** : 문서 파일 및 코드 이동
- 🪧 **[RENAME]** : 파일 이름 변경
- 🔀 **[MERGE]** : 다른 브랜치와 병합
- ♻️ **[REFACTOR]** : 전면 수정
